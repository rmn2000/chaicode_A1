##Simple HTML Resume

This project is a clean, single-page resume website built exclusively with **HTML**. 
---

### Features

* **Semantic Structure:** Uses appropriate HTML5 tags like `<header>`, `<section>`, and `<footer>`.
* **Data Organization:** Utilizes HTML tables for professional and educational history.
* **High Readability:** Employs horizontal rules and bulleted lists for a clear visual flow.


### How to Use

1. Copy the HTML code provided in the previous steps.
2. Create a new file on your computer named `index.html`.
3. Paste the code and save the file.
4. Open `index.html` in any browser to view the resume.
